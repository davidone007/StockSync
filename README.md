# StockSync App

Aplicación web para gestión de inventario y comunicación entre tenderos y proveedores.

## Características principales
- Gestión de productos y stock
- Chat entre tenderos y proveedores
- Anuncios y ofertas
- Importación de códigos de barras
- Interfaz moderna con Next.js y TailwindCSS

## Requisitos
- Node.js >= 18
- npm >= 9

## Instalación

```bash
npm install
```

## Ejecución en desarrollo

```bash
npm run dev
```

La app estará disponible en [http://localhost:3000](http://localhost:3000)

## Compilar para producción

```bash
npm run build
npm start
```

## Scripts útiles
- `npm run dev`: Ejecuta la app en modo desarrollo
- `npm run build`: Compila la app para producción
- `npm start`: Inicia la app en modo producción
- `npm run lint`: Ejecuta el linter

## Estructura del proyecto
- `app/`: Páginas y lógica principal
- `components/`: Componentes reutilizables
- `data/`: Archivos de datos (ej. CSV, JSON)
- `lib/`: Utilidades
- `public/`: Archivos públicos

## Tecnologías
- Next.js
- React
- TailwindCSS
